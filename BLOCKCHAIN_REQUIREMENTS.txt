================================================================================
                  BHOOMILEDGER BLOCKCHAIN - REQUIREMENTS
================================================================================

SYSTEM REQUIREMENTS
================================================================================
- Node.js: v18.0.0 or higher
- npm: v8.0.0 or higher
- Git (for version control)

RUNTIME DEPENDENCIES
================================================================================

Hardhat Framework:
  - hardhat v2.22.0             Ethereum development environment
  - ethers v5.x or v6.x         Ethereum library (to be installed)

DEVELOPMENT DEPENDENCIES
================================================================================
- @nomicfoundation/hardhat-toolbox  Hardhat plugin bundle
- hardhat-gas-reporter             Gas usage reporting
- solidity-coverage                 Code coverage for Solidity

SOLIDITY CONTRACTS
================================================================================
- Solidity v0.8.20
  * LandRegistry.sol            Core registry for land parcels
  * LandTransfer.sol            Land transfer escrow logic
  * LandLoan.sol                DeFi loan management
  * LandInheritance.sol         Will and inheritance management

INSTALLATION
================================================================================

Navigate to blockchain directory:
  cd blockchain

Install dependencies:
  npm install

ENVIRONMENT SETUP
================================================================================

Create .env file in blockchain directory with:

  Wallet:
  WALLET_PRIVATE_KEY=your_private_key_here
  WALLET_ADDRESS=0x8540Bc44e748d1bC90E3965136E596F1963713d9

  Network:
  POLYGON_AMOY_RPC_URL=https://rpc-amoy.polygon.technology
  POLYGONSCAN_API_KEY=your_polygonscan_api_key

  Deployment:
  NETWORK=amoy

COMPILATION
================================================================================

Compile all contracts:
  npx hardhat compile

View compiled artifacts:
  ls artifacts/contracts/

Expected Output:
  - LandRegistry.json
  - LandTransfer.json
  - LandLoan.json
  - LandInheritance.json

DEPLOYMENT
================================================================================

Deploy all contracts (requires ~0.6 POL):
  npx hardhat run scripts/deploy.cjs --network amoy

Individual contract deployment:
  npx hardhat run scripts/deploy-registry.cjs --network amoy
  npx hardhat run scripts/deploy-transfer.cjs --network amoy
  npx hardhat run scripts/deploy-loan.cjs --network amoy
  npx hardhat run scripts/deploy-inheritance.cjs --network amoy

SAVING CONTRACT ADDRESSES
================================================================================

After successful deployment, add addresses to backend/.env:

  LAND_REGISTRY_ADDRESS=0x...
  LAND_TRANSFER_ADDRESS=0x...
  LAND_LOAN_ADDRESS=0x...
  LAND_INHERITANCE_ADDRESS=0x...

TESTING
================================================================================

Run all tests:
  npx hardhat test

Run tests with coverage report:
  npx hardhat coverage

KEY CONTRACT FUNCTIONS
================================================================================

LandRegistry Contract:
  - registerParcel()             Register new land parcel
  - transferOwnership()          Transfer parcel ownership
  - getParcelDetails()           Fetch parcel information

LandTransfer Contract:
  - initiateTransfer()           Start escrow transfer
  - verifyDocuments()            Verify transfer documents
  - completeTransfer()           Finalize transfer
  - cancelTransfer()             Cancel transfer

LandLoan Contract:
  - createLoan()                 Create loan against parcel
  - recordRepayment()            Record EMI payment
  - calculateEMI()               Calculate EMI amount
  - foreclose()                  Foreclose on defaulted loan
  - clearMortgage()              Clear paid-off mortgage

LandInheritance Contract:
  - createWill()                 Create inheritance will
  - executeWill()                Execute will on condition
  - revokeWill()                 Revoke active will
  - addBeneficiary()             Add beneficiary to will
  - removeBeneficiary()          Remove beneficiary from will
